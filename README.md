# Favicon Files for LumberMargin.com

## Files included:
- `favicon.ico` — Multi-size ICO (16, 32, 48px) — place in root
- `favicon-16x16.png` — Browser tab icon
- `favicon-32x32.png` — Browser tab icon (retina)
- `favicon-48x48.png` — Windows site icon
- `favicon-64x64.png` — Windows site icon (larger)
- `favicon-96x96.png` — Google TV / Android shortcut
- `favicon-128x128.png` — Chrome Web Store
- `apple-touch-icon.png` — iOS home screen (180x180)
- `android-chrome-192x192.png` — Android home screen
- `android-chrome-512x512.png` — Android splash screen
- `mstile-150x150.png` — Windows tile
- `site.webmanifest` — PWA manifest
- `browserconfig.xml` — Microsoft browser config

## Add this to your HTML `<head>`:

```html
<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="msapplication-TileColor" content="#228B22">
<meta name="theme-color" content="#228B22">
```

## Setup:
Place all files in the root of your site (or adjust paths accordingly).
